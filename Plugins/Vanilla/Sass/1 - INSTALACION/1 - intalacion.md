###### ===--- Instalacion por (npm) ---=== ######
### ========================================= ###

<!-- Para instalar (Sass) de manera global debemos ejecutar el siguiente comando:  -->

```bat
	npm install -g sass

	: Permite ver la version de (Sass).
	sass --version

	: Proporciona la lista de comandos que podemos utilizar.
	sass -v
```
