### ========================================= ###
###### ===--- Instalacion por (npm) ---=== ######
### ========================================= ###

<!-- Luego de instalar Node JS, colocamos el comando para instalar Ionic. -->

```bat
	: Instalamos de manera global.
	npm i -g @ionic/cli
```

<!-- Comprobamos que Ionic esta instalado. -->

```bat
	ionic -v
	: Alternativamente para ver la version de ionic instalada.
	ionic --version
```
