### ================================ ###
###### ===--- jsonwebtoken ---=== ######
### ================================ ###

Es una libreria que se utiliza para verificar y validar (JSON Web Tokens [JWT]).

### ========================================= ###
###### ===--- Instalacion por (npm) ---=== ######
### ========================================= ###

<!-- Instalacion via (npm). -->

```bat
	npm install jsonwebtoken
```

<!-- Ahora importamos. -->

```js
	const jwt = require("jsonwebtoken");
```
