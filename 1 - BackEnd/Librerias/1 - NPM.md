### ======================================== ###
###### ===--- Instalacion de (NPM) ---=== ######
### ======================================== ###

Primero debes haber instalado [](NodeJS).

```bat
	node -v
```

Una vez instalado [](NodeJS) procede con la instalacion de NPM.

```bat
	: Instala la ultima version de (NPM) de manera global.
	npm install -g npm
```
